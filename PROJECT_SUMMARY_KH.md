# рҹ“ҡ LearnCore LMS вҖ” бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ„бҹ’бһҒбҹҒбһ”бһӮбһҳбҹ’бһҡбҹ„бһ„

LearnCore бһӮбһәбһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶ (LMS) бһҠбҹҸбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷ бһ“бһ·бһ„бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ№бҹҮбһҹбҹ’бһҗбһ¶бһ“бһҹбһ·бһҖбҹ’бһҹбһ¶бҹ” бһңбһ¶бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһңбҹҒбһ‘бһ·бһҖбһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһ”бҹӮбһ„бһ…бҹӮбһҖбһҹбһ·бһ‘бҹ’бһ’бһ·бһҸбһ¶бһҳбһҸбһҪбһ“бһ¶бһ‘бһёбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ (Role-based) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһ·бһҖбҹ’бһҹбһ¶ бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһҳбҹҒбһҡбҹҖбһ“ бһңбһҢбҹ’бһҚбһ“бһ—бһ¶бһ–бһҹбһ·бһҹбҹ’бһҹ бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ–бһ·бһ“бҹ’бһ‘бһ» бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бҹҒбһүбһңбһ·бһүбҹ’бһүбһ¶бһ”бһ“бһ”бһҸбҹ’бһҡбҹ”

| **бһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ (Backend)** | PHP 8.2+, Laravel 12, Livewire v3, Filament PHP v3 |

## рҹ§© бһҳбҹүбһјбһҢбһ»бһӣ бһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһҹбҹ’бһ“бһјбһӣ

### бҹЎ. бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ“бһ·бһ„бһҹбһ·бһ‘бҹ’бһ’бһ·бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ (RBAC)
* бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһүбҹ’бһ…бһ”бҹӢ Spatie Laravel Permissionбҹ”
* бһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһ…бҹҶбһ“бһҪбһ“ бҹӨ бһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ–
  * **Super Admin / Admin:** бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ‘бһјбһ‘бҹ… бһӮбһҺбһ“бһёбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ”
  * **Teacher (бһӮбҹ’бһҡбһјбһ”бһ„бҹ’бһҡбҹҖбһ“):** бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бһ¶бһҸбҹӢбһҸбһ¶бҹҶбһ„ бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҡбһ„бһҹбҹҶбһҺбһҪбһҡ бһҖбһ¶бһҡбһ”бҹ’бһҡбһЎбһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ–бһ·бһ“бҹ’бһ‘бһ»бһҹбһ·бһҹбҹ’бһҹбҹ”
  * **Student (бһҹбһ·бһҹбҹ’бһҹ):** бһ…бһјбһӣбһҳбһҫбһӣбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбҹҒбһҡбҹҖбһ“ бһҳбһҫбһӣбһңбһҢбҹ’бһҚбһ“бһ—бһ¶бһ–бһҹбһ·бһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһ‘бһ¶бһүбһҷбһҖбһңбһ·бһүбҹ’бһүбһ¶бһ”бһ“бһ”бһҸбҹ’бһҡбҹ”

### бҹў. бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһ·бһҖбҹ’бһҹбһ¶
* бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбҹ– **бһҳбһ бһ¶бһңбһ·бһ‘бҹ’бһҷбһ¶бһӣбҹҗбһҷ вһ” бһҠбҹҒбһ”бҹүбһ¶бһҸбһәбһҳбҹүбһ„бҹӢ вһ” бһ”бһ“бҹ’бһ‘бһ”бҹӢбһҹбһ·бһҖбҹ’бһҹбһ¶**бҹ”
* бһҖбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҶбҹ’бһ“бһ¶бҹҶбһҹбһ·бһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһҶбһҳбһ¶бһҹбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ”бһ“бҹ’бһ‘бһ”бҹӢбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бһёбһҳбһҪбһҷбҹ—бҹ”
* бһҖбһ¶бһҡбһ…бһ¶бһҸбҹӢбһҸбһ¶бҹҶбһ„бһҖбһ¶бһӣбһңбһ·бһ—бһ¶бһӮбһ”бһ„бҹ’бһҡбҹҖбһ“бһҡбһ”бһҹбҹӢбһӮбҹ’бһҡбһјбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ»бһҒбһңбһ·бһҮбҹ’бһҮбһ¶ бһ”бһ“бҹ’бһ‘бһ”бҹӢбһҹбһ·бһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһҳбҹүбҹ„бһ„бһ”бһ„бҹ’бһҡбҹҖбһ“бҹ”

### бҹЈ. бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһҳбҹҒбһҡбҹҖбһ“
* бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһ”бҹ’бһҡбһ—бҹҒбһ‘бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”
* бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбҹҒбһҡбҹҖбһ“бҹ– **бһҮбҹҶбһ–бһјбһҖ вһ” бһҳбҹҒбһҡбҹҖбһ“**бҹ”
* бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҜбһҖбһҹбһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһ‘бһҳбҹ’бһҡбһ„бҹӢбҹ– **бһҜбһҖбһҹбһ¶бһҡ (PDF) бһңбһёбһҠбҹҒбһўбһј (MP4/Streams) бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҮ (Assignments) бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“бһҹбһ·бһҖбҹ’бһҹбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—**бҹ”
* бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһңбһҢбҹ’бһҚбһ“бһ—бһ¶бһ–бһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбһ”бһҹбҹӢбһҹбһ·бһҹбҹ’бһҹбһҳбҹ’бһ“бһ¶бһҖбҹӢбҹ—бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҡбһ”бһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ—бһ¶бһӮбһҡбһҷбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ”

### бҹӨ. бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғ бһҖбһ¶бһҡбһ”бҹ’бһҡбһЎбһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ–бһ·бһ“бҹ’бһ‘бһ»
* **бһ’бһ“бһ¶бһӮбһ¶бһҡбһҹбҹҶбһҺбһҪбһҡ (Question Bank)** бһҖбһҺбҹ’бһҸбһ¶бһӣ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһҳбҹ’бһҡбһ„бһҹбҹҶбһҺбһҪбһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”
* **бһҖбһҳбҹ’бһҡбһ„бһҹбҹҶбһҺбһҪбһҡ (Quizzes) бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһЎбһ„ (Exams)** бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбһӮбҹ’бһҡбһјбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһ¶бһҸбҹӢбһҸбһ¶бҹҶбһ„бһ‘бҹ…бһҸбһ¶бһҳбһҗбҹ’бһ“бһ¶бһҖбҹӢ/бһҹбһ·бһҹбҹ’бһҹбҹ”
* бһҖбһ¶бһҡбһ”бҹ’бһҡбһӮбһӣбҹӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҮбһҸбһ¶бһҳбһўбһ“бһЎбһ¶бһү бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ–бһ·бһ“бҹ’бһ‘бһ»бҹ”
* бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ **бһңбһ·бһүбҹ’бһүбһ¶бһ”бһ“бһ”бһҸбҹ’бһҡ (Certificates)** бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹ…бһ–бҹҒбһӣбһҹбһ·бһҹбҹ’бһҹбһ”бһүбҹ’бһ…бһ”бҹӢбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷбҹ”

### бҹҘ. бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҖбҹҶбһ бһ»бһҹбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҸбһ¶бһҳ Telegram
* бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҖбҹҶбһ бһ»бһҹ (Exception Notification) бһҠбҹӮбһӣбһ…бһ¶бһ”бҹӢбһҷбһҖбһҖбҹҶбһ бһ»бһҹ бһ¬бһҖбһ¶бһҡбһ„бһ¶бһҡбһҠбҹӮбһӣбһ”бһҡбһ¶бһҮбҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһҡбһҪбһ…бһ•бҹ’бһүбһҫбһҹбһ¶бһҡбһ—бҹ’бһӣбһ¶бһҳбҹ—бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбҹ’бһҡбһ»бһҳбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶ Telegram бһҡбһ”бһҹбҹӢбһҖбҹ’бһҡбһ»бһҳбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбҹ”
